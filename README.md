### 🛰️ Featured Project: Sentry-Sat

Sentry-Sat is a self-contained educational stack that demonstrates how an edge-AI anomaly detector is integrated into a space-grade OBC. 
- **Core Engine:** C++17 OBC Simulator with Bump Allocators.
- **AI Pipeline:** TensorFlow/Python Convoluted Autoencoders quantized to TFLite INT8.
- **Security:** PUF (Physical Unclonable Function) Simulator & HMAC-SHA256 Cryptographic Signer.
